# Overview
This is the second version of my dotfiles. 

The idea is to save and document the customizations made by me to an Arch Linux and Hyprland setup, in an old and slow laptop with the objective of running smoothly yet still looking slick, modern and ready for daily use.

This version I wanted to write my own shell using qml and quickshell and focus more on learning a frontend language for a change.

Please be aware that this is still a personal project, and therefore everything is still subject to being worked on.
